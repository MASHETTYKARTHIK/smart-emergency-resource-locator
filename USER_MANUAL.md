# USER MANUAL

## Smart Emergency Resource Locator

### Introduction

Smart Emergency Resource Locator helps users quickly find nearby emergency resources such as hospitals, blood banks, police stations, and fire stations.

---

## Features

* Hospital Locator
* Blood Bank Locator
* Police Station Locator
* Fire Station Locator
* Emergency Resource Dashboard
* Resource Recommendation System

---

## System Requirements

* Python 3.10+
* Streamlit
* Pandas
* Plotly

---

## How to Run

### Step 1

Clone the repository.

### Step 2

Install dependencies.

pip install -r requirements.txt

### Step 3

Run the application.

streamlit run app.py

---

## How to Use

### Search Resources

1. Select emergency category.
2. Choose city/location.
3. Click Search.
4. View available resources.

### Dashboard

1. Open Dashboard page.
2. View statistics and analytics.
3. Monitor emergency resource distribution.
