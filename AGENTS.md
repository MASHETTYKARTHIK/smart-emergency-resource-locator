# AI Agents and Automation

This project utilizes several AI agents and automation tools to maintain high code quality and compliance.

## Agents
- **Ruff**: Fast Python linter and code formatter.
- **MyPy**: Static type checker for Python.
- **Bandit**: Security linter for Python code.
- **Semgrep**: Static analysis engine for finding bugs and security issues.
- **Spec-Kit**: Specification-driven development tool.

## Automation
- **GitLab CI/CD**: Automates the linting, testing, and deployment processes.
- **Pre-commit**: Ensures local code compliance before each commit and push.
- **Git-Cliff**: Automatically generates changelogs from conventional commits.
