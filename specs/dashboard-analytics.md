# Specification: Dashboard Analytics

## 📋 Overview
This specification defines the analytics and visualization features for the Emergency Resource Locator Dashboard.

## 🎯 Objectives
- Provide a high-level summary of available emergency resources.
- Visualize resource distribution across different areas of Hyderabad.
- Compare resource counts for better situational awareness.

## 🛠️ Requirements
1.  **Metric Cards**: Display total counts for Hospitals, Blood Banks, Police Stations, and Fire Stations.
2.  **Pie Chart**: Show the percentage distribution of different resource types.
3.  **Bar Chart**: Compare the number of resources available in each major area.

## 🧪 Verification
- Metrics must match the counts in the underlying CSV files.
- Charts must correctly reflect the filtered data when applicable.
